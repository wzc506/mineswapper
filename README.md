## 新功能：背景音乐播放

### 功能说明
- 游戏现在支持播放背景音乐功能
- 音乐文件位于 `bgm` 文件夹中
- 支持 WAV 格式的音频文件
- 支持选择指定音乐播放或随机播放
- 音乐会在一首结束后自动播放下一首

### 使用方法
1. 在游戏启动菜单中，有一个"播放背景音乐"复选框
2. 通过下拉框可以选择播放特定的音乐，或选择"随机播放"
3. 勾选该选项后开始游戏，系统将自动播放背景音乐
4. 音乐会在游戏结束时自动停止
5. 如果想再次玩游戏，可以在新的游戏菜单中重新选择是否播放音乐

### 技术实现
- 使用单例模式实现 `MusicPlayer` 类
- 使用 Java Sound API (javax.sound.sampled) 进行音频播放
- 支持随机播放和循环播放功能
- 自动检测 bgm 文件夹中的音频文件
